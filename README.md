# YouTube to MP3 Pro H

A PyQt6-based desktop application to download YouTube audio as high-quality MP3.

## Requirements
- Python 3.12.x (DO NOT use Python 3.13)
- FFmpeg installed and added to PATH
- Internet connection

## Setup (Windows)

```bash
python -m venv venv
venv\Scripts\activate
python -m pip install -r requirements.txt
python youtube_to_mp3_proh.py
